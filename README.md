TravelMate — Лабораторна робота №2

Тема: Створення структури веб-проєкту  
Проєкт: TravelMate — веб-портал для мандрівників

Структура проєкту
```
TravelMate_Lab2_Final/
  images/
    logo.jpg
    tour-1.jpg
    tour-2.jpg
  css/
  js/
  index.html
  catalog.html
  tour_details.html
  about.html
  contacts.html
  README.md
```

Сторінки
- Головна (`index.html`)
- Каталог турів (`catalog.html`)
- Сторінка туру (`tour_details.html`)
- Про нас (`about.html`)
- Контакти + форма (`contacts.html`)

Примітки
- На кожній сторінці використано семантичні елементи: `header`, `nav`, `main`, `footer`.
- Додано навігацію на всі сторінки.
- Додано базову доступність (skip-link, aria-label, коректні label для форми).
